# 対応画像フォーマット

| 対応画像                          |  拡張子  | 可否 | 備考                       |
| --------------------------------- | :------: | :--: | -------------------------- |
| ビットマップ                      |   bmp    |  可  |                            |
| JPEG                              |   jpg    |  可  |                            |
| JPEG2000                          |   jp2    |  可  |                            |
| Portable Network Graphics(PNG)    |   png    |  可  |                            |
| Tagged Image File Format (TIFF)   | tiff tif |  可  |                            |
| High Efficiency Image File Format |   heic   | 不可 | FFMPEGのアップデートに期待 |
